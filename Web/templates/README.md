# Lerchbacher Bücherdatenbank System
Digitales Bücherverwaltungssystem

## Verwendung
Dieses System soll als zentrale Übersicht über den Bücherbestand unserer Familie dienen. 
Im Moment soll der Zugriff darauf über eine Website erfolgen, später wird jedoch auch ein selbständiges Programm 
für meine Großmutter hinzukommen. Das Herzstück des Systems besteht aus einer 
SQLite Datenbank und einem Python Skript, welches Funktionen zur Verwendung der Datenbank
enthält.

## Fortschritt
Die Entwicklung des Hauptsystems (Datenbank, Zugriffsfunktionen) ist bereits abgeschlossen, wird in Zukunft
möglicherweise aber noch verbessert. Die Entwicklung am Website-Zugriffssystem ist bereits fortgeschritten und wird
zurzeit auf ca. 40 % geschätzt. Die Entwicklung am eigenständige Zugriffsprogramm wurde 
bis jetzt aber noch nicht gestartet. Der aktuelle Entwicklungsstand ist in der (Englischsprachigen) 
[Änderungsliste](../Changelog.md) einsehbar.

Der zurzeit alleinige Entwickler dieses Projekt ist Lars Lerchbacher. Dieses Projekt steht Ihnen unter Angabe der Quelle
